Programação

Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa.
Programação pode ser feita a partir uma variedade de linguagens de programação.

Empreendedorismo

O empreendedorismo é uma disciplina fundamental que capacita os alunos a desenvolverem habilidades e conhecimentos necessários para criar e gerir os seus próprios negócios. É uma matéria que estimula o pensamento crítico, a criatividade e a inovação, além de promover a compreensão dos princípios básicos dos negócios.
